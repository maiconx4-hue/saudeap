# SaúdeAP

Sistema web para gerenciamento e consulta de medicamentos das UBS do Estado do Amapá.

## Tecnologias

- Python
- Flask
- SQLAlchemy
- MySQL
- HTML
- CSS
- JavaScript

## Funcionalidades

- Cadastro de UBS
- Cadastro de Medicamentos
- Controle de Estoque
- Movimentações
- Dashboard
- Consulta Pública
